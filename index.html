Руководство по построению инфраструктуры

Добро пожаловать!

Это подробное руководство поможет вам шаг за шагом создать современную, масштабируемую и безопасную инфраструктуру для веб-приложений. Независимо от вашего уровня подготовки, вы сможете освоить использование таких технологий, как FreeBSD, Docker, Kubernetes, TypeScript, CI/CD, Prometheus и многих других.

Описание Стека Технологий.
  • FreeBSD: стабильная и безопасная операционная система для серверов.
  • Docker: позволяет контейнеризировать приложения для их независимого запуска.
  • Neo4j: графовая база данных для хранения и обработки связанных данных.
  • Kubernetes (Minikube): система оркестрации контейнеров для управления приложениями в кластере.
  • Traefik: динамический обратный прокси и Ingress Controller для маршрутизации трафика в Kubernetes с поддержкой HTTP/3.
  • GitHub Actions: платформа для автоматизации рабочих процессов CI/CD.
  • Visual Studio Code: современная среда разработки с поддержкой TypeScript и расширениями для разработки и отладки.
  • TypeScript: язык программирования, расширяющий JavaScript, с поддержкой статической типизации.
  • Gulp: инструмент для автоматизации задач разработки, таких как сборка, компиляция и обновление проекта в реальном времени.
  • Pug: шаблонизатор для HTML, позволяющий писать более чистый и понятный код разметки.
  • Sass: препроцессор CSS, добавляющий возможности переменных, вложенности и функций для более эффективной разработки стилей.
  • Prometheus и Grafana: инструменты для мониторинга и визуализации метрик.
  • ELK Stack: решения для сбора, хранения и анализа логов.
  • Velero: инструмент для резервного копирования и восстановления Kubernetes-кластеров.
  • pf (брандмауэр): для защиты сетевой инфраструктуры.
  • Lynis: инструмент для аудита безопасности системы.

Мы постарались сделать это руководство максимально понятным и доступным, чтобы вы могли не только повторить все шаги, но и понять, как и почему это работает. Начните свой путь к созданию профессиональной инфраструктуры прямо сейчас!

Примечание: Рекомендуется иметь базовые знания в области программирования и работы с командной строкой, однако даже новички смогут разобраться благодаря подробным инструкциям.

Требования и Предварительные Условия

Аппаратные Требования

  • Компьютер с Windows 11 Home Edition.
  • Минимум 16 ГБ оперативной памяти (рекомендуется 32 ГБ для комфортной работы).
  • Достаточное количество свободного дискового пространства (рекомендуется не менее 100 ГБ).
  • Поддержка виртуализации на уровне BIOS (Intel VT-x или AMD-V).

Необходимые Учетные Записи

  • GitHub: для хранения исходного кода и настройки GitHub Actions.
  • Docker Hub: для хранения Docker-образов.
  • Учетная запись Let's Encrypt (опционально): для получения SSL-сертификатов.
  • Учетная запись в облачном хранилище (AWS, GCP, Azure и т.д.): для резервного копирования с Velero.

Сетевые Требования

  • Стабильное подключение к интернету.
  • Возможность открыть необходимые порты на роутере (22, 80, 443 и другие по необходимости).
  • Доменное имя для доступа к вашему приложению.
