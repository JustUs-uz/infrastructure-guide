Введение

Роль: Разработчик

Уровень: Начинающий

Добро пожаловать в руководство для разработчика!

Это руководство предназначено для начинающих разработчиков, которые хотят освоить основные навыки и инструменты, необходимые для эффективной работы в современной IT-инфраструктуре. В ходе этого руководства вы научитесь создавать, разворачивать и управлять приложениями с использованием современных технологий и практик разработки.

Что вы узнаете

  • Настройка современной среды разработки с использованием Visual Studio Code и необходимых расширений.
  • Работа с языком TypeScript и использование преимуществ статической типизации.
  • Использование инструментов автоматизации задач разработки, таких как Gulp.
  • Создание шаблонов и стилей с помощью Pug и Sass.
  • Работа с системами контроля версий, в частности Git и GitHub.
  • Настройка CI/CD пайплайнов для автоматизации процессов сборки и деплоя.
  • Контейнеризация приложений с помощью Docker и оркестрация с Kubernetes.
  • Мониторинг и логирование приложений с использованием Prometheus, Grafana и ELK Stack.
  • Внедрение практик безопасности при разработке и развертывании приложений.
  • Проведение финальных проверок и тестирования разработанных приложений.

Предварительные требования

Для успешного освоения материала вам потребуется:

  • Базовые знания в области программирования и работы с командной строкой.
  • Желание учиться и исследовать новые технологии.
  • Доступ к компьютеру с установленной операционной системой FreeBSD или возможностью установки виртуальной машины.
  • Установленный Git для работы с системами контроля версий.

Как пользоваться этим руководством

Руководство разделено на последовательные шаги, каждый из которых посвящён определённой теме. Рекомендуется проходить шаги по порядку, чтобы обеспечить полное понимание материала.

В каждом разделе вы найдёте:

  • Описание: Общая информация о рассматриваемой теме.
  • Детальные инструкции: Пошаговые указания с командами и примерами.
  • Объяснения: Разбор используемых команд и настроек.
  • Чек-листы: Контрольные списки для проверки успешности выполнения действий.
  • Полезные ссылки: Дополнительные ресурсы для изучения темы более глубоко.
